# Capturas de pantalla del manual de ayuda

Las capturas deben generarse navegando el sistema y tomando screenshots
en resolución 1920×1080.

## Convención de nombres
- Formato: PNG
- Naming: kebab-case descriptivo
- Path: /help/{nombre}.png

## Capturas necesarias
Por cada página del sistema, capturar:
1. Vista general de la página
2. Cada sección/funcionalidad principal
3. Formularios o diálogos importantes
